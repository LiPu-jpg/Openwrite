# 项目管理功能

管理小说项目的初始化和状态查询。

## 功能

- **初始化项目**: 创建新小说项目，设置基本信息
- **项目状态**: 查看当前项目的完成度、章节数等

## 使用方式

用户说"初始化项目"或"新建项目"时，启动初始化流程。
