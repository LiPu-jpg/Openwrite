# 术语表 — 《术师手册》

> 作品设定层：《术师手册》特有的角色、术语、世界观规则。实际生成环境中，这些设定由 OpenWrite 主系统提供。此文档作为参考模板。

---

## SC1 基础术语（第1-7章）

| 术语 | 含义 | 注意事项 |
|------|------|----------|
| 术师 | 拥有超凡能力的人 | 通用称呼 |
| 术灵 | 赋予术师力量的存在 | 四柱神赐予的天使本质是"术灵"，被称为"二翼混沌术灵" |
| 翼级 | 实力等级系统 | 已知：二翼混沌（低）、三翼圣域（高）。翼有颜色：白银之翼、黄金之翼、七彩之翼 |
| 意识光幕 | 类似手机的科技设备 | 植入后颈，意念操控，有版本号（14.4.1）、芯片型号（奇迹13）、运营商（凯蒙通灵） |
| 虚境 | 由法则与知识构筑的虚幻世界 | 意识光幕功能之一，被禁止访问；术师通过术灵体内的真理之门进入 |
| 狩罪厅 | 执法机构 | 类似警察局，下辖血狂猎人 |
| 血狂猎人 | 狩罪厅的执法者 | 有猎人编号，穿白银风衣戴猎帽，持血纹长剑 |
| 记忆师 | 能调取他人记忆的职业 | 不能对接触过四柱神的人使用（记忆污染） |
| 血月审判 | 公开处刑节目 | 凯蒙第一频道播出，由全城市民参与审判 |
| 干员 | 游戏系统中的角色单位 | 亚修本人也是干员（终末观者） |
| 羁绊等级 | 干员与亚修的绑定程度 | 决定经验共享比例，0级=30% |
| 源晶 | 游戏内货币 | 用于抽卡和购买物资 |
| 荆棘 | 剑花大学的处分 | 集齐三个荆棘会被退学 |
| 剑花 | 剑花大学的校花称号 | 索妮娅被认为是67届的剑花候选 |
| 耀星 | 这个世界的太阳 | 数量随季节变化（春天两颗，夏天两颗半） |

## SC2 新增术语（第8-14章）

| 术语 | 含义 | 注意事项 |
|------|------|----------|
| 甲像 | 剑术训练用的木桩 | 核心处有十环标识，材料坚硬无比，受损后自动恢复。斩入十环=具备召唤术灵的资格 |
| 虚翼术师 | 成功召唤出术灵的术师 | 一翼术师的正式称呼 |
| 架势 | 剑术派系的修炼法/召唤法/战斗法 | 每个架势对应一种术灵。掌握架势=能召唤对应术灵 |
| 繁星架势 | 基础剑术架势 | 《剑术操典》标准架势，可召唤斩剑/切剑/刺剑三种术灵，优点是根基扎实可转修 |
| 波动架势 | 伏斯洛达家族秘传架势 | 以爆发著称，对应"波动剑"术灵 |
| 波动剑 | 一翼术灵 | 效果：发出剑气波动；被动：普通剑器攻击更容易令目标失去平衡 |
| 斩剑/切剑/刺剑 | 基础一翼剑术术灵 | 由繁星架势召唤 |
| 碎湖监狱 | 亚修被关押的监狱 | 设施豪华（影音室、情侣房间、观海天台等），有奇迹芯片控制囚犯暴力行为 |
| 奇迹芯片 | 植入后颈的控制芯片 | 检测到暴力行为时发出神经电流控制身体，在监狱和畜牧业都有应用 |
| 心灵派系 | 术师派系之一 | 可签订契约控制他人，"朋友"契约实为单方面服从关系 |
| 媚娃 | 种族之一 | 最容易精通心灵派系的种族 |
| 永生之酒俱乐部 | 政治组织/派系 | 凯蒙市政治势力之一 |
| 见切 | 索妮娅掌握的步伐技巧 | 诡异的侧滑步伐，可中断当前动作突兀地滑行回避 |

## SC3 新增术语（第15-21章）

| 术语 | 含义 | 注意事项 |
|------|------|----------|
| 贡献度 | 死刑犯的生存货币 | 初始50点，每月扣除10点，通过死斗转移。贡献度越高审判序列越后，越安全 |
| 审判序列 | 血月审判的参加顺序 | 按贡献度排名，前8名参加审判，每月一号和十五号执行 |
| 死斗社 | 监狱内的战斗组织 | 获得监狱长许可，死斗期间解除芯片限制，必须一方死亡或昨迷才结束 |
| 死斗押注 | 死斗的贡献度赌注 | 第N场必须押注N点，双方可不对等，胜者获得败者押注 |
| 苦弱术师 | 通过强化身体的体术师 | 用术灵修改血肉架构，将身体当成武器，即使禁用术灵仍保留强化效果 |
| 体术师 | 涉及锻体修炼的术师统称 | 包括剑术师、拳术师、铳术师、长枪术师、斩斧术师等 |
| 水疗 | 水术派系的治疗术灵 | 催动患者体内水分令伤口再生，会记忆优化治疗过的病症，可进化为二翼术灵 |
| 白银医疗师 | 拥有三个术灵的医疗师 | 实力较弱，治疗效果不稳定，病人更倾向找黄金医疗师 |
| 黄金医疗师 | 更高级的医疗师 | 治疗效果更好，病人首选 |
| 乌鸦面具 | 医疗师的标志性装备 | 有声音失真效果，用于威慑病人改善医患关系，也用于保密身份 |
| 言出必行 | 伊古拉的奇迹 | 由契约+共鸣+履约三个术灵组合，任何人与伊古拉达成协议后必须履行 |
| 共鸣 | 伊古拉的术灵之一 | 可与对手建立联系，获取对手的进攻意图 |
| 履约 | 伊古拉的术灵之一 | 要求对方践行自己的话语 |
| 编号牌 | 医疗师的身份标识 | 医疗师之间也以编号辨识，不知道彼此身份 |

## SC4 新增术语（第22-28章）

| 术语 | 含义 | 注意事项 |
|------|------|----------|
| 知识之海 | 虚境第一层 | 铅灰色大海+浓墨天空+白雾；航行中自动获得知识凝聚白银术力；所有术师进入虚境的第一站 |
| 时间大陆 | 虚境第二层 | 位于黄金鱼背上，二翼术师的去处，在此凝聚黄金之翼 |
| 黄金鱼 | 漂浮在知识之海的巨鱼 | 鱼背上是时间大陆，找到它可直接晋升二翼（"偷渡者"），极其罕见 |
| 真理之门 | 术灵体内的通道 | 普通人进入虚境的唯一方式，必须通过自己召唤的术灵才能找到 |
| 白银之翼 | 一翼术师的实力象征 | 由知识之海中获取的术力凝聚而成 |
| 黄金之翼 | 二翼术师的实力象征 | 在时间大陆凝聚 |
| 知识生物 | 虚境中的原生生物 | 击杀可获得术灵，爆出术灵不固定但基本同派系，会随现实新知识更新 |
| 泥鱼龙 | 虚境常见知识生物 | 长翅膀的巨鱼，会喷水炮和飞行冲锋，击杀可获得水术/土术派系术灵 |
| 术师手册（世界观设定） | 术师死后灵魂自动生成的书 | 术师的"墓碑"，记录术师一生最重要的事，可吸收获得随机技艺，有污染性（三观不合会损伤灵魂） |
| 术师投影 | 术师死后灵魂返回虚境形成的实体 | 守护传承小岛，击败后消散，留下术师手册和术灵 |
| 传承小岛 | 虚境中漂浮的小岛 | 术师执念的衍化，取走传承后沉入海底 |
| 连射 | 一翼术灵 | 射击后可即刻进行第二次射击，被时代淘汰的铳术术灵，最低收购价1白银币；限制：需远程射击工具 |
| 反侦察精通 | 亚修从狩猎笔记吸收的技能 | 丛林猎人+战争军人的生存技能 |
| 冒想室 | 术师探索虚境的专用静室 | 剑花大学施展了奇迹【星光逐影】加速思维+吸引知识（又称"冥想楼"） |
| 星光逐影 | 剑花大学的冒想楼奇迹 | 加快思维速度，接引繁星法主的星光令灵魂更容易吸引知识 |
| 术灵喂养 | 术灵需要货币维持存在 | 一翼术灵喂白银币，二翼术灵喂黄金币，不喂养会自行消散 |
| 专精术灵 | 对术师有特定要求的术灵 | 如波动剑要求用剑，连射要求有远程射击工具 |
| 通用术灵 | 对术师无特定要求的术灵 | 如"鹰眼"只要有眼睛就能催动 |
| 偷渡者 | 在知识之海找到黄金鱼的术师 | 跳过正常晋升流程直接成为二翼术师 |
| 繁星十大杰出剑圣 | 繁星国度的剑术师排名 | 特洛赞教授入选，被称为"隐手剑圣" |
| 特洛赞 | 三翼剑术师，索妮娅的导师 | "隐手剑圣"，有望冲击四翼传奇，从未公开收过学徒 |
| 席林·多尔 | 幕后人物 | 朗拿提及的名字，与瓦尔卡斯挑战亚修有关，狱卫称其为"席林先生" |
| 斩我 | 剑术防御奇迹 | 一翼白银奇迹，净化异常状态，可清除奇迹芯片 |
