# 通用节奏技法 (Universal Rhythm & Pacing)

> 通用技法层：跨作品通用的节奏控制原则。

## 段落长度分布原则

| 类型 | 典型占比 | 使用场景 |
|------|----------|---------|
| 短段（1-2句） | 40-60% | 动作、对话、情绪冲击、转折点 |
| 中段（3-5句） | 30-40% | 场景描写、心理展开、设定说明 |
| 长段（6句+） | 10-20% | 密集信息、背景铺陈、思维奔涌 |

**核心原则**：短段建立节奏基调，长段制造沉浸或冲击，两者交替使用。

## 章节长度光谱

| 类型 | 字数范围 | 适用风格 |
|------|----------|---------|
| 微章节 | 500-1000字 | 快节奏、悬疑、网文 |
| 短章节 | 1000-2000字 | 轻小说、日更连载 |
| 标准章节 | 2000-4000字 | 传统网文、类型小说 |
| 长章节 | 4000-8000字 | 严肃文学、史诗奇幻 |

## 开头模式库

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 直入场景 | 无铺垫，直接进入动作或对话 | 快节奏、动作戏、悬疑 |
| 环境速写 | 1-2句环境描写后立刻进入事件 | 场景转换、氛围建立 |
| 悬念钩子 | 以问题或反常现象开头 | 章节开头、吸引注意 |
| 情绪延续 | 承接上章结尾的情绪状态 | 连续剧情、情感戏 |
| 时间锚点 | 明确时间信息开头 | 时间跳跃后、多线叙事 |

## 结尾模式库

| 模式 | 说明 | 效果 |
|------|------|------|
| 悬念钩子 | 引出新元素或新问题 | 驱动继续阅读 |
| 反转/打脸 | 刚建立的期望被颠覆 | 惊喜感、记忆点 |
| 情绪余韵 | 留下意味深长的一句话 | 情感沉淀 |
| 暗示关联 | 暗示多线的联系 | 伏笔感、整体感 |
| 冷静宣言 | 简短宣言暗示变化即将到来 | 期待感 |
| 反差心声 | 角色的反差内心 | 喜剧效果 |

## 核心节奏模型

### 短→长→短 波浪模型

```
短段群（建立基调）→ 长段（信息/情感高峰）→ 短段群（释放/过渡）
```

**关键约束**：长段不宜连续超过2-3段，否则节奏"塌陷"。

### 紧张-松弛循环

**基础模式**：紧张 → 释放 → 新紧张

**变体**：
| 模式 | 结构 | 适用场景 |
|------|------|---------|
| 先甜后打脸 | 建立期望 → 颠覆 | 喜剧、反转 |
| 绝望-希望-行动 | 展示困境 → 找到出路 → 执行 | 冒险、成长 |
| 渐进式震撼 | 层层叠加 → 高潮爆发 | 慢热型、史诗场景 |
| 冒险-教学循环 | 行动 → 间隙说明 → 行动 | 设定密集型 |

## 信息密度控制

### 分散原则

- 设定说明不集中堆砌，分散在角色行动和思考中
- 每个信息点后给读者"消化时间"（反应、吐槽、动作）
- 密集信息段后必须有节奏缓冲

### 乒乓球规则

信息提供者每说1-2个信息点，接收者必须有反应：
- 追问
- 吐槽
- 表情/动作变化
- 内心独白

**禁忌**：连续灌输超过3段无反应。

## 场景切换技法

| 切换类型 | 技法 | 示例 |
|----------|------|------|
| 章节间 | 直接切换，新章节=新场景/视角 | 章节结束即切换 |
| 章内软切 | 分隔符（空行、省略号、星号） | 「……」单独成行 |
| 多线切换 | 过渡语+地理/时间锚点 | "与此同时，在..." |
| 时间跳跃 | 明确时间标记或暗示 | "三天后"、"当夜幕降临" |

## 节奏诊断清单

- [ ] 长段是否连续超过3段？
- [ ] 信息灌输是否超过3段无反应？
- [ ] 章节开头是否在3句内进入核心？
- [ ] 章节结尾是否有钩子或余韵？
- [ ] 紧张段落后是否有释放？
