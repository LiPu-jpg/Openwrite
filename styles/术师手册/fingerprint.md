# 风格指纹 (Style Fingerprint)

> 作品风格层：提取自《术师手册》。此风格仅适用于本作品。

## 一句话概括

轻松幽默现代感吐槽为核心基调的轻小说。

## 核心标识（优先级排序）

1. **吐槽密度与质量**：几乎每个严肃设定后都跟吐槽，用现代视角解构奇幻。核心特征是"歪楼"——吐槽落点经常与触发点不在同一维度，故意跑题制造笑点
2. **表里不一角色塑造**：角色外在表现与内心独白形成强烈反差（外冷内慌、外甜内毒）
3. **叙述者-角色声音融合**：叙述句中直接使用角色口头禅、思维方式和比喻体系，读者分不清哪句是叙述、哪句是内心独白
4. **现代感语言**：网络用语、游戏术语、职场梗自然融入叙述，不生硬
5. **克制煽情**：用细节传递情感，煽情后接冷静转折
6. **设定自然嵌入**：通过角色行动/阅读/对话引入设定，不大段灌输。每个信息点都有角色反应（"乒乓球"节奏）
7. **双线叙事信息差**：读者比角色多一层信息，制造悬念和期待
8. **旁观者视角链**：通过多个旁观者的递进式反应展现主角实力，而非主角自述。信息密度极高
9. **先甜后打脸的反转节奏**：先建立期望再颠覆，也有"先绝望再找希望"的反向模式
10. **旁观者误读**：旁观者对主角行为的错误解读制造喜剧效果，同时推动剧情
11. **行动先于解释**：角色先行动再理解后果，而非先分析再行动
12. **不解释的世界观细节**：随手丢出不解释的小细节制造世界观厚度感，触发时机以"角色思维流的自然延伸"为主

## 风格DNA维度

| 维度 | 值 | 说明 |
|------|-----|------|
| 幽默类型 | 解构 > 自嘲 > 歪楼 > 反差 | 解构是第一驱动力，歪楼是区分度最高的标识 |
| 叙述距离 | 极近 | 第三人称但高度贴近角色意识，几乎等同第一人称内心独白 |
| 情感表达 | 克制 | 用细节和反差传递情感，拒绝直接煽情 |
| 信息密度 | 高 | 旁观者视角链、双线信息差、设定嵌入行动中 |
| 段落偏好 | 短段60% | 1-2句短段为基本节奏单元 |
| 章节长度 | 800-1500字 | 轻小说标准短章节 |
| 现代感浓度 | 极高 | 职场/游戏/互联网/商业/科技五大领域全覆盖 |
| 吐槽密度 | 极高 | 几乎每个严肃设定后都跟吐槽 |
| 叙述者介入 | 隐性 | 不直接评论，通过角色口吻渗入叙述实现 |
| 视角切换 | 章节边界 | POV切换在章节边界或明确场景转换处 |

## 风格反模式（生成时避免）

- 中性叙述句（丢失角色化用词）
- 对比式段子代替联想跳跃型歪楼
- 复制原文已有细节而非自主创造不解释细节
- 角色行动前过度分析（应"先做再想"）
- 连续灌输设定超过3段无角色反应
- 煽情段落无冷静转折收尾
