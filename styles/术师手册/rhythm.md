# 节奏风格 (Rhythm & Pacing)

> 作品风格层：提取自《术师手册》。此风格仅适用于本作品。

## 段落长度分布：60/30/10

| 类型 | 占比 | 使用场景 |
|------|------|---------|
| 短段（1-2句） | 60% | 战斗动作、内心吐槽、对话、戏剧性转折/冲击 |
| 中段（3-5句） | 30% | 场景描写、设定说明、角色心理展开 |
| 长段（6句+） | 10% | 密集世界观设定、角色背景、角色"一口气想通一件事"的思维奔涌、NPC连续讲解 |

## 章节长度

800-1500字/章（轻小说标准短章节）。

## 标题风格

简短直白，通常是一句话概括或角色名，带有轻微的剧透/悬念感。
> 「我被逮捕了」「我真的是坏人」「术师手册」「观者」「试炼」「玻璃舍友情」「来源：术师手册」

## 开头模式

- **直接进入场景**，无铺垫：
  > 「"名字？""亚修……亚修·希斯。"」「来源：术师手册」
- **简短环境描写后立刻进入事件**：
  > 「这里是一个小岛。」「来源：术师手册」

## 结尾模式

| 类型 | 说明 |
|------|------|
| 悬念钩子 | 引出新元素或新问题 |
| 反转/打脸 | 刚建立的期望被颠覆 |
| 情绪余韵 | 留下意味深长的一句话 |
| 暗示关联 | 暗示双线的联系 |
| 反转前的冷静宣言 | 简短宣言暗示反转即将到来 |
| 反差心声 | 角色的反差内心作为喜剧钩子 |

## "短→长→短"核心节奏

大量短段建立轻快基调→突然插入一个信息密集的长段制造冲击/沉浸→立刻回到短段（通常是吐槽或对话）恢复轻快感。

**关键约束**：长段不能连续出现超过2-3段，否则节奏会"塌"。

## 信息密度分布

- 设定说明不集中堆砌，分散在角色行动和思考中
- 设定说明后通常紧跟吐槽，避免读者感到枯燥
- 游戏界面信息使用「」括起来的格式化文本，与叙述文本视觉区分
- 设定说明的"乒乓球"节奏：信息提供者每说1-2个信息点，接收者就必须有反应，不能连续灌输超过3段

## 紧张-松弛核心模式

**基础模式**：紧张→打脸→吐槽→新紧张

**变体模式**：
| 模式 | 说明 |
|------|------|
| 先甜后打脸 | 先建立期望再颠覆（SC1核心） |
| 绝望铺垫→希望之光→行动 | 先展示绝望再找出路（SC3） |
| 渐进式震撼 | 时间推移+旁观者反应层层叠加构建高潮（SC2），慢热型 |
| 冒险→教学→冒险循环 | 战斗间隙自然引入设定（SC4虚境） |

## 场景切换方式

- **章节间**：直接切换，新章节开头通常是新场景、新视角
- **章内**：使用省略号分隔（「……」单独成行）表示时间跳跃或场景转换
- **双线切换**：使用"与此同时"等过渡语，配合地理/时间信息
- **时间跳跃**通常很短（几小时到一天）
